# YouTube AI Automation Agent

This project automates the process of creating YouTube videos based on trending news. It scrapes news articles, generates video scripts, creates SEO-optimized content, and manages the uploading and scheduling to YouTube.

## Features
- News scraping from Google News
- Content generation (scripts, titles, hashtags)
- AI-generated thumbnails and voiceovers
- YouTube API integration for uploads
- Automated posting and analytics monitoring

## Installation
1. Clone the repository
2. Install the dependencies listed in `requirements.txt`
3. Set up your API keys in the `.env` file
4. Run the `src/main_agent.py` to start the automation
